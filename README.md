# Українізатор для Hitman: Codename 47
Hitman: Codename 47 Ukrainian Localization

![example workflow](https://github.com/romamakar/hitman_codename_47_ukr/actions/workflows/dotnet-desktop.yml/badge.svg?branch=master)

Це текстовий українізатор для гри Hitman: Codename 47

Для встановлення скачайте файл Hitman_Codename_47_Ukr_Installer.exe з https://github.com/romamakar/hitman_codename_47_ukr/releases/download/exe/Hitman_Codename_47_Ukr_Installer.exe та запустіть та дотримуйтесь вказівок інсталятора.


Буду дуже радий підтримці, кожен охочий може докластись до розробки та виправляти. Всі стрічки знаходяться в папці xml.
